# College Student Management System (SQL)

## Project Description
This project is a simple SQL-based database system used to manage student information in a college.

The system stores details like:
- Student ID
- Student Name
- Age
- Course
- Marks
- Department

## Technologies Used
- MySQL
- SQL
- Command Prompt

## Features
- Create Database
- Create Table
- Insert Student Records
- Retrieve Data using SELECT
- Filter Data using WHERE
- Update Student Records
- Delete Records
- Perform Aggregate Functions

## Example Queries

CREATE DATABASE college_db;

CREATE TABLE students(
stud_id INT,
stud_name VARCHAR(50),
age INT,
course VARCHAR(50),
marks INT,
department VARCHAR(50)
);
